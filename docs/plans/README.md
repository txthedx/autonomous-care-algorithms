# Phase plans

This directory holds the per-phase implementation plans for the CDS engine
(Phases 1–6 in [docs/architecture.md](../architecture.md)).

Each `phase-N.md` is produced with **ultraplan** on Claude Code on the web,
reviewed in the browser, and imported here. A cloud Routine then implements the
phase from the approved plan and opens a PR for human merge — see
[docs/autonomous-setup.md](../autonomous-setup.md).
